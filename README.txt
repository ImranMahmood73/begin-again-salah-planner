PWA notes:
- Open the GitHub Pages link on a phone.
- Use Add to Home Screen / Install App.
- PWA service workers normally require HTTPS or localhost. GitHub Pages provides HTTPS.

Privacy model:
- User progress is saved locally in the browser.
- Backup and restore uses a JSON file controlled by the user.
- No account or cloud database is used.

Scope:
- First version is for adult Muslim men estimating ordinary missed five-daily prayers.
- Women’s calculations require a separate pathway because menstruation and postnatal bleeding affect the estimate.
